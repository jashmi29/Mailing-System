A desktop-based Java application simulating an email client locally.
Built with Java Swing, MySQL, and JDBC.

**Features**

- User registration & secure login

- Compose, send, and receive emails

- Inbox & sent mail management

- File attachments

- Search & filter emails

**Technologies**

- Java (JDK 8+)

- Java Swing

- MySQL

- JDBC

- NetBeans IDE

Installation

- Install Java JDK 8+ and MySQL Server

- Configure the database

- Import MySQL Connector/J/J

- Run the project in NetBeans IDE or via the JAR file
